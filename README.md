# MirrorPlay 🦋
### 鏡像解密英文學習工具

> 解密遊戲，不是考試。慢慢來。

![Uploading IMG_2092.png…]()



## 這是什麼？

一個把英文文字左右翻轉成鏡像的小工具。
你看著鏡像文字，用手寫出正確的英文。
沒有分數、沒有批改、沒有壓力。

## 為什麼做這個？

我有 ADHD。學英文對我來說最大的障礙不是英文本身，
而是傳統學習工具帶來的壓力和無聊。

MirrorPlay 的設計基於三個原則：

**數位降噪** — 睡前不想滑社群焦慮，
但又想做點有意義的事。這個工具夠安靜。

**微啟動** — 截圖或貼上文字就開始，
不需要先選課程、設目標、做測驗。三秒進入狀態。

**拿回主導權** — 沒有演算法告訴你該學什麼，
你自己選想解密的內容，想停就停。

## 功能

- 🪞 上傳圖片 OCR 或直接貼文字 → 自動鏡像翻轉
- ✍️ 手寫解密（支援壓感筆）
- 🔍 點擊查看圖片、中文翻譯、AI 語境解釋
- 🔊 例句朗讀
- 🎨 防躁塗鴉板（注意力的安全閥）
- 🎲 抽 5 個隨機複習
- 📱 PWA 支援，可加到手機主畫面

## 使用方式

1. 開啟 [mirror-play.vercel.app](https://mirror-play.vercel.app)
2. 首次使用需輸入你的 Anthropic API Key（AI 功能用）
3. 上傳含英文的圖片，或直接貼上英文文字
4. 開始解密！

> AI 功能需要 [Anthropic API Key](https://console.anthropic.com/)，
> 沒有 Key 也能使用核心的鏡像 + 手寫功能。
